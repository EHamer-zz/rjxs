# Eric Hamer's comments
This webapp uses the node rxjs package to provide subscriptions through a Subject, as well as observables, to track the navigation of the International Space Station. This app also uses the of, timer, throwError, catchError, tap, and map features from rxjs.  The data is updated every ten seconds.
